=============================================================================================
Keyword Suggestor - Project based on Trie DS
=============================================================================================
This is a project, where  user can add words, check if something is a valid word 
and delete the word. 

Build Instructions :

1) First Build the Make file from CMakeLists.txt using below command on the project path-
	cmake -S . -B build
2) Go to Build directory and run below command to compile the code
	make clean;make
